# UICS 音频批量生成使用说明

本说明介绍如何在 **UICS** 中使用「音频生成」功能，从 Excel 的 **音频汇总** 表批量生成配音（火山引擎 / ComfyUI Qwen3-TTS）。

---

## 1. 前置准备

- 已正确部署并启动 UICS（访问 `http://localhost:5000` 可打开页面）。
- 已准备好剧本 Excel：默认文件为 `UICS/all_episodes.xlsx`，包含 **音频汇总**、**角色汇总** 等工作表。
- 若使用 **外部 TTS 服务**：
  - 火山引擎：已在控制台创建应用并获取 `appid` 与 `access_token`。
  - ComfyUI Qwen3-TTS：已启动 ComfyUI，并在本机或可访问服务器上配置好 `Qwen3-TTSVoiceCloneAPI.json` 工作流。

---

## 2. Excel 表结构与必填列

音频批量生成依赖 `音频汇总` 与 `角色汇总` 两个工作表（表名以实际 Excel 为准，这里采用推荐命名）。

### 2.1 音频汇总（主表）

常见关键列示例（字段名与你现有表保持一致即可）：

- **剧集id**：如 `EP01`，用于剧集过滤。
- **分镜号**：如 `EP01_SQ01`，与图像/视频使用相同分镜号。
- **角色**：说话人名称（需能在角色汇总中找到对应记录）。
- **音频id**：生成音频文件名前缀，例如 `EP01_SQ01_A01`。
- **音频内容**：要合成的文本（台词）。
- 其他列（如备注、情感等）可按需要扩展。

> 实际列名以你当前 Excel 为准；UICS 读取时会通过 `excel_reader` 中的模型进行字段映射。

### 2.2 角色汇总（用于默认音色/参考音频）

典型列：

- **角色名**：与音频汇总中「角色」匹配。
- **参考音色 / 参考音频**（可选）：可存放参考音频文件名或路径，用于 ComfyUI Qwen3-TTS 的语音克隆。

角色信息会在生成时用于：

- 为火山引擎选择不同的 `voice_type`（如果你在配置中做了映射）。
- 为 Qwen3-TTS 选择对应角色的参考音频。

---

## 3. 配置生成器（非 ComfyUI 模式）

UICS 支持两种音频生成后端：

- **火山引擎 TTS**：`generator_type = "volcengine"`（外部 WebSocket API）
- **ComfyUI Qwen3-TTS**：`generator_type = "comfyui"`（本地或远程 ComfyUI）

### 3.1 火山引擎（推荐先配置）

1. 在 `UICS/lib` 目录中，编辑已经创建好的示例配置：

   文件：`UICS/lib/audio_generator_config.json`

   ```json
   {
     "appid": "YOUR_APPID_HERE",
     "access_token": "YOUR_ACCESS_TOKEN_HERE",
     "endpoint": "wss://openspeech.bytedance.com/api/v1/tts/ws_binary"
   }
   ```

2. 用你在火山引擎控制台获取的实际值替换：
   - `appid`
   - `access_token`
   - 如无特殊要求，可保留默认 `endpoint`。

3. UICS 在调用音频生成时，会按以下优先级读取配置：
   - 接口显式传入的 `appid` / `access_token` / `endpoint`
   - 请求体中的 `config_path` 指向的 JSON
   - 默认路径中的 `audio_generator_config.json`：
     - `UICS/lib/audio_generator_config.json`
     - `~/.audio_generator_config.json`
     - `./audio_generator_config.json`

### 3.2 ComfyUI Qwen3-TTS（数字人声）

1. 启动 ComfyUI，确认本机可以访问（例如 `http://127.0.0.1:8188`）。
2. 将 `Qwen3-TTSVoiceCloneAPI.json` 放到 UICS 的 `lib` 目录（已随项目自带）。
3. 确认参考音频文件已放入 **ComfyUI 的 input 目录**（通常是 `ComfyUI/input`），或在 UICS 内上传：
   - 在 UICS 的音频生成流程中，我们已实现自动上传参考音频至 ComfyUI 的 `input` 目录，并在工作流的 `LoadAudio` 节点更新为对应文件名。

## 4. 在 UICS 页面操作批量生成

1. 打开浏览器访问 `http://localhost:5000`，登录 UICS。
2. 切换到 **「Excel编辑」** 标签页：
   - 点击 **「加载Excel」**，选择工作表 **音频汇总**，确认数据无误。
3. 切换到 **「媒体生成」** 标签页，找到 **「音频生成」** 区块。

在音频生成区块中，你可以配置：

- **输出目录**：通常保持 `./output`，后端会强制使用 `UICS/output`。
- **剧集过滤**（可选）：如 `EP01`，只生成该剧集的音频。
- **分镜过滤**（可选）：如 `EP01_SQ01`，只生成单个分镜；支持留空表示全部。
- **生成器类型**：
  - 选择 **火山引擎**：使用外部 TTS（`generator_type=volcengine`）。
  - 选择 **ComfyUI (Qwen3-TTS)**：使用 ComfyUI 的 Qwen3-TTS 语音克隆。
- **ComfyUI 服务器**（仅 ComfyUI 模式）：例如 `127.0.0.1:8188`。
- **工作流文件路径**（仅 ComfyUI 模式，可选）：默认 `Qwen3-TTSVoiceCloneAPI.json`。

4. 配置完成后点击 **「开始生成音频」**。

5. 打开 **「任务管理」** 标签页，可查看：
   - 任务类型：`audio`
   - 进度百分比与状态（running / completed / failed）
   - 错误信息（如配置不正确会提示缺少 appid / access_token 等）。

6. 生成成功后：
   - 音频文件会保存在 `UICS/output` 下，按 `音频id` 命名，例如：  
     `EP01_SQ03_A01_00002_.flac` 或 `.wav`。
   - 在 **图像汇总表** 中，「音频预览」列会按分镜号自动匹配对应音频文件，并在前端以 `<audio controls>` 播放。

## 5. 通过 API 调用批量生成（可选）

如果你希望用脚本/工具直接调用后端，也可以使用 REST API。

### 5.1 火山引擎示例（`generator_type=volcengine`）

请求：`POST /api/generate/audio`  
请求头：需要携带 `X-Session-ID`（登录后页面里已经自动处理，脚本调用时可从浏览器复制）。

```json
{
  "output_dir": "./output",
  "episode_filter": "EP01",
  "shot_filter": null,
  "generator_type": "volcengine",
  "config_path": null
}
```

- 若 `config_path` 为空，则使用上面提到的默认 `audio_generator_config.json`。

### 5.2 ComfyUI Qwen3-TTS 示例（`generator_type=comfyui`）

```json
{
  "output_dir": "./output",
  "episode_filter": "EP01",
  "shot_filter": "EP01_SQ03",
  "generator_type": "comfyui",
  "config_path": null,
  "comfyui_server": "127.0.0.1:8188",
  "workflow_path": "Qwen3-TTSVoiceCloneAPI.json"
}
```

## 6. 常见问题排查

- **提示缺少 appid / access_token：**
  - 检查 `UICS/lib/audio_generator_config.json` 是否存在且已填写。
  - 或接口是否正确传入 `config_path`/显式参数。

- **生成成功但前端音频预览没有更新：**
  - 确认输出文件在 `UICS/output` 下，文件名包含分镜号（例如 `EP01_SQ03_A01_00002_.flac`）。
  - 页面在收到任务完成通知后会自动刷新 Excel；如果仍没看到最新预览，可手动点击「加载Excel」重新加载。

- **ComfyUI 模式下音频生成失败：**
  - 检查 ComfyUI 是否已启动、服务器地址是否正确。
  - 确认参考音频文件已上传到 ComfyUI `input` 目录，且 LoadAudio 节点的输入为正确的文件名。

如需进一步定制（例如为不同角色配置不同情感、映射火山引擎的 `voice_type`），可以查看：

- `UICS/lib/audio_generator.py`
- `UICS/docs/使用数据说明.md` 中「三、音频生成」章节
