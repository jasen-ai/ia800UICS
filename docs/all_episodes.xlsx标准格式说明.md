# `all_episodes.xlsx` 标准格式说明

本文档给出 UICS 运行所需的 `all_episodes.xlsx` 工作表与列名“标准写法”，用于保证 Excel 导入、预览以及图片/视频/音频生成逻辑可正常工作。

标准列名以 `UICS/lib/excel_reader.py` 的 `ExcelDataReader.export_to_excel()` 导出的字段为准。

## 1. 工作表列表

UICS 的 Excel 读取器默认读取以下工作表（sheet name 必须一致）：

- `音频汇总`
- `角色汇总`
- `场景汇总`
- `图像汇总`（兼容旧名：`图像提示词`）
- `剪辑汇总`

另外，UICS 前端在读取时会“额外显示”预览列（只读），不要求你在 Excel 里手动创建：

- `图像预览`
- `首帧预览`
- `末帧预览`
- `视频预览`
- `场景图预览`

## 2. 通用约定

### 2.1 ID 字段类型

`场景id` / `角色id` 支持两种形式：

- 数字（例如 `1`）
- 字符串（例如 `LOC-01`、`CHAR-8EBF5C5E`）

当导入时，这两个字段能根据你 Excel 中使用的列名大小写/空格变体进行兼容解析，但建议你仍按下文标准列名填写。

### 2.2 参考图字段格式

- `图像汇总`.`参考图`：支持填空或填 1+ 个参考图信息；用于“图生图(reference_images 1~3)”或“图生图/文生图”分支选择。
- `场景汇总`.`参考图`：同样支持填空或填多张参考图信息（用于“场景生成”的图生图分支）。

参考图字段通常会被当作“字符串”，可使用逗号分隔多项（也兼容中文分隔符/空白分隔，具体以代码解析为准）。

## 3. 各工作表标准列

### 3.1 `音频汇总`

| 列名 | 含义 | 类型/示例 |
|---|---|---|
| `剧集id` | 所属剧集 | `EP01` |
| `分镜号` | 分镜标识（与其他表的分镜号一一对应） | `EP01_SQ01` |
| `场景内容` | 场景文本内容（给音频/提示词使用） | `...` |
| `场景图片提示词` | 场景图片/视觉相关提示词 | `...` |
| `剧情角色` | 叙事/口播角色（文本） | `...` |
| `音频角色` | TTS/配音所用角色（文本） | `...` |
| `音频情感` | 音频情绪 | `开心/悲伤/...` |
| `音频id` | 音色/声音映射 ID | `TTS-001` |
| `音频图片id` | 数字人相关的图片/视觉 ID（如果你使用数字人链路） | `...` |
| `音频内容` | 要生成/播放的音频文本 | `...` |
| `时长` | 音频时长（秒） | `3.2` |
| `输出视频id` | 对应输出视频的标识（若用于数字人/剪辑关联） | `...` |
| `备注` | 备注信息 | `...` |
| `参考样本` | 参考样本（用于音色克隆等扩展） | `...` |

### 3.2 `角色汇总`

| 列名 | 含义 | 类型/示例 |
|---|---|---|
| `角色名` | 角色显示名 | `林婉` |
| `角色id` | 角色 ID（用于文件名/参考图解析） | `CHAR-8EBF5C5E` 或 `1` |
| `音频id` | 该角色对应的音频/音色 ID | `TTS-001` |
| `参考音色` | 音色参考（可选） | `...` |
| `角色描述` | 角色描述 | `...` |
| `视觉特征` | 视觉特征（提示词扩展使用） | `...` |
| `图像提示词` | 角色图像提示词（图生图/提示词扩展使用） | `...` |
| `出现章节` | 出现章节（可选） | `第1章` |
| `出现次数` | 出现次数（可选） | `12` |
| `出现剧集及分镜号` | 出现位置汇总（可选） | `EP01_SQ01,EP01_SQ05` |
| `备注` | 备注信息（可选） | `...` |
| `参考图` | 角色参考图信息（用于图生图 reference_images 中的参考图2/3） | `...` |

兼容说明：代码在读取 `参考音色` 时会优先使用 `参考音色`，若为空会退回使用 `参考图`；读取 `参考图` 时若为空，会尝试 `参考图.1` 或 `参卡图` 作为兼容列名。

### 3.4 `图像汇总`（主表）

`图像汇总` 用于驱动图片生成与视频生成（其中 `视频提示词` 用于视频）。

| 列名 | 含义 | 类型/示例 |
|---|---|---|
| `剧集id` | 所属剧集 | `EP01` |
| `分镜号` | 分镜标识（与文件名/预览查找一一对应） | `EP01_SQ01` |
| `场景名` | 场景名（用于解析参考图1：场景id → 场景图目录的 `{场景id}.png/.jpg`） | `咖啡厅内景` |
| `场景内容` | 场景文本内容 | `...` |
| `角色` | 角色名（用于解析参考图2：角色id → 角色图目录的 `{角色id}.png/.jpg`，并可能参与提示词扩展） | `林婉` |
| `镜头类型` | 镜头类型（可选） | `近景/远景/...` |
| `图像提示词` | 参考图（图像提示词）生成用提示词 | `...` |
| `首帧提示词` | 首帧生成用提示词 | `...` |
| `末帧提示词` | 末帧生成用提示词 | `...` |
| `视频提示词` | 视频生成用正提示词 | `...` |
| `备注` | 备注信息（可选） | `...` |
| `参考图` | 图生图 reference_images 的“参考图3”（以及扩展解析剩余参考图） | `...` |

兼容说明：若你的 Excel 没有 `图像汇总` 工作表，但有旧名 `图像提示词`，UICS 会自动兼容读取 `图像提示词`。

## 4. 与输出文件/预览列的关系（快速记忆）

- 图片生成（图像汇总的分镜号）输出一般是：`{分镜号}_ref_00001.png`、`{分镜号}_first_00001.png`、`{分镜号}_last_00001.png`
- 视频预览/播放会通过 `{分镜号}_video_*.mp4` 等文件匹配
- 场景生成会输出：`{场景id}.png`，并在 `场景汇总` 中展示为 `场景图预览`
