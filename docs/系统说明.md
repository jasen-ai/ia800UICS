# UICS 系统说明

## 一、系统概述

UICS（Unified Intelligent Content System）是一个基于 **FastAPI** 的 Web 应用，用于多用户协同编辑剧本 Excel，并驱动**图片、视频、音频**的批量生成与任务管理。支持 ComfyUI、Sora、火山引擎、Nano Banana、Qwen3-TTS 等后端，并支持数字人视频（图+音频 s2v）与数字人声生成。

## 二、技术栈

### 后端

| 技术 | 用途 |
|------|------|
| FastAPI | Web 框架、REST API、依赖注入 |
| Uvicorn | ASGI 服务器 |
| Pandas / OpenPyXL | Excel 读写 |
| Passlib / python-jose | 密码加密与认证 |
| WebSocket | 实时任务状态、Excel 变更通知 |

### 前端

| 技术 | 用途 |
|------|------|
| Vue 3 | 页面与状态管理 |
| Handsontable | 在线表格编辑 |
| 原生 WebSocket | 与后端长连接 |

### 媒体生成依赖（按功能选用）

- **图片**：ComfyUI（z_image_client）、Nano Banana
- **视频**：ComfyUI（图生视频 i2v/i2vse/s2v）、Sora
- **音频**：火山引擎 TTS、ComfyUI（Qwen3-TTS 数字人声）

## 三、功能模块

### 3.1 认证与用户

- 登录、注册、登出
- 会话通过 `X-Session-ID` 请求头传递
- 角色：admin（可看所有任务）、user（仅本人任务）
- 用户数据默认存于 `users.json`（可改为数据库）

### 3.2 Excel 编辑

- 读取 `all_episodes.xlsx` 全部工作表
- 在线查看、编辑（Handsontable），支持多表切换
- 写入单元格、添加行、删除行
- 通过 WebSocket 广播变更，多端同步

### 3.3 媒体生成

| 功能 | 说明 | 后端/工作流 |
|------|------|-------------|
| 图片生成 | 按图像提示词、参考图等生成 | ComfyUI / Nano Banana，可选参考图、首末帧 |
| 视频生成 | 按视频提示词、首帧（及可选末帧）生成 | ComfyUI i2v/i2vse 或 Sora |
| 数字人视频生成 | 单图 + 对应分镜音频生成口型视频 | ComfyUI s2v（act_video_wan2_2_14B_s2v-aigc-api.json） |
| 音频生成 | 按音频汇总与角色生成配音 | 火山引擎 或 ComfyUI (Qwen3-TTS) |
| 数字人声生成 | 固定使用 Qwen3-TTS 语音克隆 | ComfyUI |

生成任务在后台线程执行，状态与进度经 WebSocket 推送到前端；完成后可刷新预览（图片/首末帧/视频等）。

使用 ComfyUI 时，需先启动 ComfyUI 并在页面填写「ComfyUI 服务器」（如 `127.0.0.1:8188`）。ComfyUI 的启动方式、API 监听、工作流与模型清单见：[ComfyUI 工作流与模型](ComfyUI工作流与模型.md)。

### 3.4 任务管理

- 创建任务：图片/视频/音频/数字人视频/数字人声
- 任务状态：pending → running → completed / failed
- 进度百分比、结果与错误信息
- 任务列表按用户与角色过滤

### 3.5 预览与输出

- 输出目录默认：`UICS/output`
- 预览接口：`/api/preview/{filename}`，按分镜号查找参考图、首帧、末帧、视频
- 生成完成后通过 WebSocket 通知前端刷新预览

## 五、运行依赖加载逻辑

- 若存在 `UICS/lib/` 且其中存在 `excel_reader.py`：**优先从 `UICS/lib` 加载**所有生成器相关模块与工作流 JSON，便于整体拷贝部署。
- 否则：从**项目根目录**（050tool）加载，适合在仓库内开发运行。

## 六、主要 API 概览

### 认证

- `POST /api/auth/login`：登录  
- `POST /api/auth/register`：注册  
- `POST /api/auth/logout`：登出  

### Excel

- `GET /api/excel/read`：读取整表  
- `POST /api/excel/write`：写入单元格  
- `POST /api/excel/add_row`：添加行  
- `POST /api/excel/delete_row`：删除行  

### 生成任务

- `POST /api/generate/image`：图片生成  
- `POST /api/generate/video`：视频生成  
- `POST /api/generate/audio`：音频生成  
- `POST /api/generate/audio_digital`：数字人声生成  

### 任务与预览

- `GET /api/tasks`：任务列表  
- `GET /api/tasks/{task_id}`：任务详情  
- `GET /api/preview/{filename}`：预览文件（图片/视频等）  

### WebSocket

- 连接：`ws://host:5000/ws`  
- 事件：`join_task` / `leave_task`、`task_update`、`refresh_preview`、`excel_updated` 等  

详细请求体与响应格式见 **http://localhost:5000/docs**（Swagger）。

## 七、配置项说明（server.py）

| 变量 | 含义 | 默认 |
|------|------|------|
| `BASE_DIR` | UICS 根目录 | 当前 server.py 所在目录 |
| `EXCEL_FILE` | Excel 路径 | `{BASE_DIR}/all_episodes.xlsx` |
| `OUTPUT_FOLDER` | 生成文件输出 | `{BASE_DIR}/output` |
| `SECRET_KEY` | 会话/安全密钥 | 环境变量或默认值，生产请修改 |
| `EXCEL_FILE` / `OUTPUT_FOLDER` | 见上 | 可按部署需要修改 |

## 八、扩展与维护

- **新增生成器类型**：在 `server.py` 中增加对应 API 与后台任务函数，并调用现有 `excel_reader` / `image_generator` / `video_generator` / `audio_generator` 的批量接口。
- **新增 Excel 工作表**：无需改代码，读取时已包含所有 sheet；前端需能切换并展示对应列。
- **单独部署**：执行 `copy_deps.py` 后整体拷贝 `UICS` 目录，详见 [DEPLOY.md](../DEPLOY.md)。
