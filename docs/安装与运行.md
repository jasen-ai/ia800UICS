# UICS 安装与运行

## 一、环境要求

- **Python**：3.8 及以上
- **系统**：Windows / Linux / macOS

## 二、安装

### 2.1 获取 UICS 目录

- 在项目内：使用 `UICS` 目录即可。
- 单独部署：先执行依赖复制（见 [DEPLOY.md](../DEPLOY.md)），再将整个 `UICS` 目录拷贝到目标机器。

### 2.2 安装 Python 依赖

在 **UICS 目录**下执行：

```bash
cd UICS
pip install -r requirements.txt
```

主要依赖包括：

- `fastapi`、`uvicorn[standard]`：Web 服务
- `python-multipart`：表单与文件上传
- `pandas`、`openpyxl`：Excel 读写
- `passlib[bcrypt]`、`python-jose[cryptography]`：认证与密码

### 2.3 准备 Excel 与目录

- 将剧本 Excel 放到 UICS 目录下，命名为 `all_episodes.xlsx`（或修改 `server.py` 中 `EXCEL_FILE` 指向你的文件）。
- 确保存在 `output`、`uploads` 目录（首次启动可自动创建）。

### 2.4 单独部署时复制运行依赖（可选）

若要将 UICS **整体拷贝到其他机器**运行，需先把运行依赖复制到 `UICS/lib/`：

在**项目根目录**（050tool）执行：

```bash
python UICS/scripts/copy_deps.py
```

之后将整个 `UICS` 目录（含 `lib/`）拷贝到目标机，在目标机安装上述 Python 依赖后即可运行。详见 [DEPLOY.md](../DEPLOY.md)。

## 三、运行

### 3.1 启动方式

在 **UICS 目录**下任选一种：

**方式一：直接运行**

```bash
python server.py
```

**方式二：使用 uvicorn（推荐）**

```bash
uvicorn server:app --host 0.0.0.0 --port 5000
```

开发时可加 `--reload`：

```bash
uvicorn server:app --host 0.0.0.0 --port 5000 --reload
```

**方式三：使用启动脚本（Linux/macOS）**

```bash
chmod +x start.sh
./start.sh
```

### 3.2 访问与默认账号

- 浏览器打开：**http://localhost:5000**（本机）或 **http://<服务器IP>:5000**（远程）。
- 默认管理员：用户名 `admin`，密码 `admin123`。
- 新用户可在登录页点击「注册」创建。

### 3.3 端口与地址

- 默认监听：`0.0.0.0:5000`（所有网卡）。
- 若端口被占用，可修改 `server.py` 末尾或 uvicorn 的 `--port`，例如：`--port 8000`。

## 四、基本使用（界面操作）

### 4.1 登录与 Excel 编辑

- 浏览器打开上述地址后，使用默认账号或注册账号登录。
- 点击「Excel 编辑」→「加载 Excel」，选择工作表后可在线编辑；修改会自动保存。

### 4.2 媒体生成（图片 / 视频 / 音频）

- 点击「媒体生成」标签页，按需选择**图片生成**、**视频生成**、**音频生成**或**数字人声生成**。
- 配置项说明：
  - **输出目录**：生成文件保存位置（UICS 会使用服务端配置的 `OUTPUT_FOLDER`，一般无需改）。
  - **剧集过滤** / **分镜过滤**：可选，填写后只处理指定剧集或分镜（多个分镜用逗号分隔）。
  - **生成器类型**：图片可选 ComfyUI 或 Nano Banana；视频可选 ComfyUI 或 Sora；音频可选火山引擎或 ComfyUI。
- **使用 ComfyUI 时**：在对应区域的「ComfyUI 服务器」中填写地址，格式为 `IP:端口`。
  - 本机：`127.0.0.1:8188`（需先在 ComfyUI 目录执行 `python main.py`）。
  - 远程：`<ComfyUI 机器 IP>:8188`（ComfyUI 需加 `--listen` 启动）。详见 [ComfyUI 工作流与模型](ComfyUI工作流与模型.md) 中「如何开启 ComfyUI API 服务」。
- 点击「开始生成」后，到「任务管理」查看进度与结果；生成完成后可刷新预览查看图片/视频等。

### 4.3 任务管理与预览

- 「任务管理」中可查看所有任务状态（pending / running / completed / failed）、进度与错误信息。
- 预览通过输出目录下的文件提供；若生成失败，请根据任务错误信息排查（如 ComfyUI 未启动、地址错误、缺模型等）。

## 五、运行前检查

| 检查项 | 说明 |
|--------|------|
| Python 版本 | `python --version` 或 `python3 --version` 为 3.8+ |
| 依赖安装 | 在 UICS 目录执行 `pip install -r requirements.txt` 无报错 |
| Excel 文件 | `UICS/all_episodes.xlsx` 存在（或已修改 `EXCEL_FILE`） |
| 单独部署 | 若为拷贝部署，已执行 `copy_deps.py` 且保留 `UICS/lib/` |
| ComfyUI（可选） | 使用图片/视频/数字人视频/数字人声生成时，需先启动 ComfyUI 并在页面填写「ComfyUI 服务器」地址（如 `127.0.0.1:8188`），详见 [ComfyUI 工作流与模型](ComfyUI工作流与模型.md) |

## 六、基本使用

1. **登录**：浏览器访问 UICS 地址，使用默认账号或注册账号登录。
2. **Excel 编辑**：在「Excel 编辑」中点击「加载 Excel」，选择工作表后可直接在表格中编辑，修改会自动保存。
3. **媒体生成**：在「媒体生成」中选择要生成的内容（图片/视频/音频/数字人声），按需选择生成器类型；若选 ComfyUI，需在「ComfyUI 服务器」中填写已启动的 ComfyUI 地址（如 `127.0.0.1:8188` 或 `192.168.1.100:8188`）。可填写剧集过滤、分镜过滤缩小范围，再点击对应「开始生成」。
4. **任务与预览**：在「任务管理」中查看任务列表与实时进度；任务完成后可在预览区域查看生成的图片/视频等。生成文件默认输出到 UICS 的 `output` 目录。

更多界面与流程说明见 [QUICKSTART.md](../QUICKSTART.md)；ComfyUI 的启动方式、模型准备与工作流说明见 [ComfyUI 工作流与模型](ComfyUI工作流与模型.md)。

## 七、常见问题

**Q: 启动报错找不到 `excel_reader`、`video_generator` 等模块？**

- 在项目内运行：确保从「项目根目录」或「UICS 目录」启动，且未删除上层目录中的对应 `.py`。
- 单独部署：必须先执行 `python UICS/scripts/copy_deps.py`，并保留生成的 `UICS/lib/` 再拷贝。

**Q: 端口 5000 已被占用？**

- 使用 `uvicorn server:app --host 0.0.0.0 --port 8000` 换端口，或关闭占用 5000 的进程。

**Q: 图片/视频/音频生成失败？**

- 图片/视频（ComfyUI）：确认 ComfyUI 服务已启动且地址、端口与页面配置一致。
- 音频（火山引擎）：确认已配置对应 API/密钥。
- 查看「任务管理」中该任务的错误信息及控制台日志。

**Q: Excel 打不开或保存失败？**

- 确认 `EXCEL_FILE` 路径正确且进程有读写权限；检查磁盘空间。

更多功能与接口说明见 [系统说明](系统说明.md)。
